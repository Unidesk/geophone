Geophone API server

Geophone is a very simple API server that delivers the LibPhoneAPI library via a REST interface, created and hosted by Unidesk Corporation (www.unidesk.com). This is a private API server, but the source code is published on GitHub at Unidesk/geophone if you wish to host your own server.

To use GeoPhone, the API end-point is /geophone/API?key=XYZ where XYZ is a license key according to the license key file keys.xml.

APIs

  • /geophone/lookup?phone=XXXXXXXX&key=XYZ
  • /geophone/batch&key=XYZ
